# Introduction

## Burni FHIR Server
Burni is an implementation of the FHIR server with Node, Express, and MongoDB providing very simple ways to generate Mongoose Schema, API code and customize the <a href="https://www.hl7.org/fhir/">HL7 FHIRÂ® specification</a> Currently, Burni support both Windows and Linux environment to enable developers to rapidly deploy a FHIR service.

Burni use version 4.0.1 (R4) of the HL7 FHIR specification.
