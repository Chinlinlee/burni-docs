## Burni FHIR Server
Burni is an implementation of the FHIR server with Node, Express, and MongoDB providing very simple ways to customize the <a href="https://www.hl7.org/fhir/">HL7 FHIRÂ® specification</a> Currently, Burni support both Windows and Linux environment to enable developers to rapidly deploy a FHIR service.
