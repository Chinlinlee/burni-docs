# 搜尋參數概觀
Burni 目前支援的 [FHIR search operations](http://www.hl7.org/fhir/search.html) 如下:
## Search Parameter Types
- Number
- Date/DateTime
- String
- Token
- Reference
## Parameters for all resources
- _id
- _lastUpdated
## Search result parameters
- _count
